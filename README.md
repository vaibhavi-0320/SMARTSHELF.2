# SmartShelf - NFT Book Marketplace

A decentralized NFT book marketplace built on Ethereum blockchain (Sepolia testnet).

## Features

- **NFT Book Publishing**: Authors can publish books as NFTs
- **Blockchain Ownership**: Books are stored as NFTs with verifiable ownership
- **IPFS Storage**: Book content hosted on decentralized IPFS via Pinata
- **MetaMask Integration**: Connect wallet to buy and sell books
- **Royalty System**: Authors receive royalties on secondary sales

## Tech Stack

- **Frontend**: React, TypeScript, Vite, Tailwind CSS
- **Blockchain**: Ethereum (Sepolia Testnet), Solidity Smart Contracts
- **Storage**: IPFS via Pinata
- **Wallet**: MetaMask, RainbowKit, Wagmi
- **Backend**: Supabase (Database, Authentication)
